# 用游标分页稳定返回大列表

偏移量分页在数据持续写入时很容易出现重复或漏读。这个示例用“最后一条记录的位置”生成游标，让下一页从稳定位置继续读取，更适合时间倒序列表、消息流和审计记录接口。

## 设计要点

- 游标只暴露编码后的字符串，不让客户端直接拼 SQL 偏移量
- 排序键使用 `(CreatedAt, ID)`，避免同一秒多条记录时不稳定
- `Paginate` 返回 `nextCursor`，没有下一页时为空字符串

## 示例说明

这里用内存切片模拟数据库结果集，真实场景中可把游标解析后映射为 SQL 条件：`created_at < ? OR (created_at = ? AND id < ?)`。

## 运行方式

```bash
go test ./daily/2026/06/29/cursor_pagination
```
