# 用滑动窗口统计延迟 SLO

只看平均延迟很容易掩盖尾延迟问题。这个示例用固定时间窗口维护最近一段时间的请求延迟，计算成功率和 P95，便于把接口表现直接映射到 SLO 观测面板。

## 设计要点

- 只保留窗口期内样本，避免历史数据稀释当前问题
- 用阈值判断请求是否“达标”，计算成功率
- 同时输出 P95，补足平均值对尾延迟不敏感的问题

## 示例说明

这个结构适合嵌在轻量级限流器、边车或业务内自监控逻辑里。真正接 Prometheus 时，可以把统计结果定时导出成 gauge 或 summary。

## 运行方式

```bash
go test ./daily/2026/07/01/latency_window
```
